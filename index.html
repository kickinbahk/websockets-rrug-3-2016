# Adding ES2015 ## to # Your Rails App
## 2 gems are Necessary 

                    #Gemfile
                    gem "sprockets"
                    gem "sprockets-es6"
#### Add to top of `application.js` ``` require 'sprockets/es6' ```
### Install Presets ``` npm install babel-preset-es2015 --save-dev ```
### Create `.babelrc` config and enable Plugin(s) ``` echo '{ "presets": ["es2015"] }' > .babelrc ```
## Es6 functionality is added to any `.es6` file
